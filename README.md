# VIABILIDAD DE GENERACIÓN DE ENERGÍA CON FUENTES FOTOVOLTAICAS ANTE VARIACIONES CLIMATICAS
## 📖 Descripción del Proyecto
El objetivo principal es analizar la viabilidad y el despliegue global de la energía solar a gran escala. Utilizando datos de instalaciones operativas, este proyecto aplica un ciclo completo de analítica de datos: desde el diseño conceptual y relacional de la base de datos, hasta la extracción de *insights* mediante SQL y la visualización interactiva en una aplicación web.

El proyecto cumple con la estructura requerida de entrega individual, integrando MER, modelo relacional, scripts SQL, consultas analíticas, y una aplicación web funcional.

## 🛠️ Tecnologías y Herramientas Utilizadas
* **Base de Datos:** MySQL (Diseño MER y Scripts DDL/DQL).
* **Procesamiento de Datos:** Python (Pandas).
* **Visualización y Web App:** Streamlit, Plotly Express.
* **Control de Versiones:** Git & GitHub.

## 📂 Estructura del Repositorio
* `solar_weatherN.mwb`: Diagramas conceptual y lógico de la arquitectura de datos.
* `solar_weather.sql`: Script de creación de tablas (DDL) y definición de llaves primarias/foráneas.
* `Landing_Page`/ `app.py`: Aplicación principal en Streamlit que contiene la Landing Page corporativa y el Dashboard analítico.
* `datos_solares.csv`: Dataset principal procesado para alimentar las visualizaciones.
* `Landing_Page`/ `requirements.txt`: Listado de librerías y dependencias de Python.
* `proyecto solar_weather 2.0.pdf`: Documento técnico
